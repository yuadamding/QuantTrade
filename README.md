# QuantTrade

## RL Quant Framework

The reinforcement-learning trading framework lives in `rl_quant/`.

It includes:

- intraday QQQ DQN workflows,
- daily strategy allocation workflows,
- hourly and minute causal-transformer allocation workflows,
- market-data bootstrap scripts for daily/hourly/minute OHLCV datasets.

See `rl_quant/README.md` for commands and data-format details.
